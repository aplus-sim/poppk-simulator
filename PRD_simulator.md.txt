   # PRD — PopPK 파라미터 추출 & PK 프로파일 시뮬레이터

    ## 1. 개요 (Overview)
    POPPK 논문 PDF에서 핵심 PK 파라미터 표를 정확히 추출해 표준화된 CSV로 만들고,
    그 CSV를 source로 하는 **단일 HTML 기반 웹 PK 프로파일 시뮬레이터**를 제공한다.
    설치/서버 없이 브라우저에서 즉시 실행 가능해야 한다.

    ## 2. 배경 & 문제 (Background)
    - 2단 레이아웃 논문 PDF는 `pdftotext`의 `-raw`/`-layout` 출력이 서로 어긋나 표 값 신뢰성이 낮다.
      - 따라서 추출은 **페이지 이미지 렌더링 → 시각적 전사·교차검증**을 표준 절차로 한다.
      - 추출된 파라미터로 임상가가 투여요법별 농도-시간 프로파일과 환자 간 변동성을 빠르게 탐색할 수단이 필요하다.

      ## 3. 목표 (Goals)
      - G1. 지정 표(`Final pharmacokinetic parameter estimates` / `Parameter estimates`)에서
        **parameter / Estimate / RSE**를 정확히 추출.
      - G2. 약물 열을 포함한 **단일 통합 CSV** 산출.
      - G3. CSV를 읽어 **Monte-Carlo 변동성(중앙값 + 5–95% 밴드)** 포함 PK 프로파일을 그리는 단일 HTML 앱.

    ### Non-Goals
      - PD/efficacy(CRP·DAS28) 완전 동역학 시뮬레이션(향후 확장).
      - 다수 mAb 라이브러리(`PPKPARAMETER.csv`) 전체 연동(향후 확장).

    ## 4. 사용자 (Users)
      임상약리/PK-PD 연구자, 모델러 — NONMEM 결과를 빠르게 시각 탐색하려는 사용자.

      ## 5. 범위 (Scope)
    ### 5.1 데이터 추출
      - 대상:
        - `Eculizumab_POPPK.pdf` → **Table 3** (eculizumab, 2-구획 IV)
