## ② 상세 카드

### 🔧 PS-0 — 3중 복사본 단일화
- **현재**: `index.html` = `pk_simulator.html` = `web/index.html` 3벌이 완전 동일
- **문제**: 한 곳만 고치면 나머지가 뒤처져 어느 게 최신인지 혼란
- **목표**: 소스 1개로 통일, 나머지는 사본/동기화(배포는 `web/`)
- **작업**: 정본 파일 1개 지정 → 나머지 정리 → vercel 배포 경로 확인
- **검증**: 정본 수정 후 배포본과 내용 일치

### 🔍 PS-A — 현행 covariate 반영 여부 검증
- **현재**: 약물별 `build()`에서 체중·성별만 하드코딩("합리적 근사"라고 코드에 명시됨)
- **문제**: covariate가 실제로 곡선에 반영되는지 미확인
- **목표**: 체중·성별 변경이 CL/V·곡선에 반영되는지 사실 확인
- **작업**: 체중 80→100kg 등으로 바꿔 곡선 변화 관찰, `build()` 코드 검토
- **검증**: 입력 변경 시 곡선이 예측대로 이동하면 확인 완료 → 결과가 **PS-3로 이어짐**

### 🔧 PS-1 — 데이터 주도 엔진 + 69종 라이브러리 연동
- **현재**: `MODELS` 객체에 eculizumab·adalimumab **2종을 코드로 하드코딩**, `extracted_poppk_params.csv`만 읽음
- **문제**: 약물 추가 시 매번 코드 수정 필요, `PPKPARAMETER.csv`의 **69종(모델 160개)** 미연동
- **목표**: CSV 컬럼에서 구조를 자동 판별해 드롭다운에 69종 노출, 코드 수정 없이 CSV 추가만으로 확장
- **작업**:
  1. `PPKPARAMETER.csv` 파서
  2. 컬럼→구조 매핑(VP/Q 유무→구획수, ka 유무→흡수, Route→IV/SC)
  3. 하드코딩 `MODELS` 제거 → generic `build()`
- **검증**: 드롭다운 69종 표시 + 임의 5종 프로파일이 합리적 범위 + **기존 2종 baseline 회귀 없음**(eculizumab Cmax≈387, adalimumab Cmax≈9.0)

### 🔧 PS-2 — 비선형 소실(MM/TMDD) 반영
- **현재**: 모든 약물을 **선형 소실**(k10=CL/Vc)로만 계산
- **문제**: 라이브러리 **76개 모델이 TMDD/NonLin**으로 표시되고 `Km`/`Vm` 값이 있는데 미사용 → 저농도 곡선 부정확
- **목표**: 플래그+Km/Vm 있는 약물은 MM 소실항 `−Vm·C/(Km+C)` 적용
- **작업**:
  1. `deriv`에 MM 항 추가
  2. 플래그 기반 선형/비선형 분기
  3. UI에 모델 유형 표기
- **검증**: 비선형 약물 곡선이 저농도에서 더 빠르게 감소하는지 정성 확인

### 🔧 PS-3 — 공변량 UI 일반화
- **현재**: 체중·성별만 지원, 약물별 하드코딩
- **문제**: 약물마다 cov가 다름(CL_WT·VC_WT·CL_SEX·CL_ALB, size descriptor BW/BSA/FFM). "어떤 항목·값을 선택해 시뮬할지" UI에 없음
- **목표**: CSV의 cov 컬럼을 읽어 **해당 약물에 존재하는 cov만** 입력칸 자동 생성, Ref size 기준 정규화
- **작업**:
  1. cov 컬럼 파싱
  2. 약물별 동적 폼 생성
  3. `build()`에서 cov 일반 적용
- **검증**: 약물 전환 시 cov 입력칸이 데이터대로 바뀌고, 변경이 CL/V에 반영

### 🔧 PS-4 — PD/efficacy 시뮬 (CRP·DAS28) *(선택)*
- **현재**: PK 농도만 시뮬, adalimumab PD 파라미터(kin·kout·C50·IC50·DAS28_0) 미사용
- **문제**: 효능 지표(CRP, DAS28) 시뮬 불가
- **목표**: adalimumab에 CRP 간접반응 + DAS28 Imax 모델 추가, PK와 함께 표시
- **작업**:
  1. 간접반응 ODE 추가
  2. PD 축/탭
  3. IIV 반영
- **검증**: CRP가 투여 후 감소 곡선을 그리고 문헌과 정성 일치
